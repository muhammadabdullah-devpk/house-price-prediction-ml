# 🏠 House Price Prediction using Machine Learning

## 🚀 Overview
This project is a Machine Learning-based regression system that predicts house prices using different property features such as area, location, number of rooms, and other relevant attributes. The goal is to demonstrate end-to-end machine learning workflow using Python.

## 🧠 Problem Statement
Real estate prices depend on multiple complex factors. This project builds a predictive model that learns patterns from historical data and estimates house prices accurately using regression techniques.

## 📊 Dataset
The dataset is provided in Excel format:
- File: `task6_house_price_dataset.xlsx`
- Contains property-related features used for training the model.

## ⚙️ Workflow
The project follows a complete ML pipeline:
- Data loading
- Data cleaning
- Exploratory Data Analysis (EDA)
- Feature selection & engineering
- Model training
- Model evaluation
- Price prediction

## 🧠 Machine Learning Model
- Algorithm: Regression (Linear Regression / similar models)
- Library: Scikit-learn
- Evaluation Metrics:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score

## 🛠 Tech Stack
- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
